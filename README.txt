Disk File System Simulation

Input order options are following.
0.Delete the disk.
1.Print all files in the disk and the content of all of them.
2.Format the disk ,user entering he's block size and the direct enteris.
3.Creating new file.
4.Open file.
5.Close file.
6.Write to a file.
7.Read from file.
8.Delete file.

For running the program ,you need to enter following command.
Compile:	g++ -std=c++0x -g main.cpp  -o task
Run: 		task


DISK_SIM_FILE.txt is disk simulation file.
